ESP32 CSI Toolkit

The ESP32 CSI Toolkit provides researchers access to Channel State Information (CSI) directly from the small ESP32 microcontroller. An ESP32 flashed with this toolkit can allow for online CSI processing from a computer, smartphone or even standalone. These features along with the ESP32's small size can give researchers the ability to perform tasks such as Wi-Fi Sensing with CSI in new ways. All without requiring complicated firmware updates all packaged as a concise codebase.

Progress on writing tool documentation are currently in progress. Codebase and documentation will soon appear on this website.

Features of Toolkit

  • Transmit CSI through USB serial in real-time to computer or smartphone
  • Record CSI to on-board microSD card for later processing
  • Active Access Point (CSI-Receiver)
  • Active Station (CSI-Receiver)
